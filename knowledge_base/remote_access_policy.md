# Remote Access Policy

Employees must enroll a primary device in MFA before using the VPN or cloud admin tools.

If an employee loses a phone or hardware token, they must open a helpdesk request and complete identity verification before MFA can be reset.

The IAM team can issue a temporary bypass code that expires after 24 hours.

High-risk admin accounts must re-enroll with a managed device before privileged access is restored.

Access to production systems is blocked after five failed MFA attempts within one hour.
